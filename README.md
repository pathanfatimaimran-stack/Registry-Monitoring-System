# Windows Registry Change Monitoring System

## 📌 Description
This project detects unauthorized changes in Windows Registry using baseline comparison and monitoring techniques.

## 🎯 Objectives
- Monitor registry keys
- Detect unauthorized changes
- Generate alerts
- Log suspicious activity

## 🛠️ Tools Used
- Python
- File handling

## ▶️ How to Run
python3 registry_monitor.py

## 📊 Output
- Alerts on registry changes
- Logs saved in registry_log.txt
